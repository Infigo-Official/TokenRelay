# 1. Get your certificate in PEM format (usually from your CA or via openssl)
# 2. Create a file with .crt extension (e.g., my-ca.crt)
# 3. Paste the certificate content (including the BEGIN/END lines)
# 4. Save the file in this directory
# 5. Restart the container: docker-compose restart

# How to export a certificate from a website:

# How to convert DER to PEM:
# openssl x509 -inform der -in certificate.cer -out certificate.crt

# How to view certificate details:
# openssl x509 -in certificate.crt -text -noout
